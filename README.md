# 🎯 Event Automation Platform

AI-powered event automation platform built using Python agents, Streamlit, HubSpot, LinkedIn automation, and email workflows.

### 🤖 AI Agents

* Copywriter/content generation agent

* LinkedIn outreach agent
* Follow-up automation agent
* Venue recommendation & outreach agent
* Registration management agent
* HubSpot CRM integration agent

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/khushbooyd-coder/event-automation.git
cd event-automation
```

---

### Create Virtual Environment

```bash
python -m venv .venv
source .venv/bin/activate
```

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_key
HUBSPOT_API_KEY=your_key
EMAIL_PASSWORD=your_password
EMAIL_ADDRESS=your_email
```

---

## ▶️ Run Application

### Streamlit Dashboard

```bash
streamlit run dashboard.py
```
